**************************
PRE REQUISITES
**************************

1. R should be installed and accessible by the command line. Install all the packages required by R
2. A6Prediction-0.0.1-SNAPSHOT-jar-with-dependencies.jar, A6Prediction-0.0.1-SNAPSHOT.jar, and all the files required are provided in folder A6Run
3. Make sure you have pandoc and latex installed

**********************************
RUNNING PROGRAM USING MAKE FILE
**********************************
We were not able to run the project on cluster so have executed the project in pseudo cluster mode.
To run the project simply go to the A6_Run folder containing all the files needed to run the project from terminal. Make changes to makefile to add path of the folders in hadoop pseudo mode.
Start hadoop. give command make prediction to run the project on pseudo cluster mode.
